## 🎲 Tenzi Game (React)

An interactive implementation of the classic Tenzi dice game built with React. Players strategically roll and hold dice to match all values, combining simple mechanics with dynamic state management and responsive UI updates.

### ✨ Features
- 🎮 Fully interactive dice rolling and holding mechanics
- ⚛️ State-driven UI using React hooks
- 🔁 Real-time game updates and win detection
- 🧠 Strategic gameplay (choose which dice to hold)
- 💡 Clean, modular component structure
- 🚀 Deployed with GitHub Pages

### 📜 How to Play
1. Roll until all dice show the same number.
2. Click a die to freeze its value between rolls.
3. Continue rolling the remaining dice until all match.
4. Win by matching all dice!

### 🛠 Tech Stack
- React (Functional Components & Hooks)
- JavaScript (ES6+)
- HTML5 & CSS3
